# jokey-cli

A tiny Python command‑line tool that fetches a random joke from the
[Official Joke API](https://official-joke-api.appspot.com/) and prints it
to the terminal.

## Features

- Zero‑config installation (just `pip install -r requirements.txt`).
- Works on Python 3.9‑3.11.
- Fully tested and linted with GitHub Actions CI.

## Usage

```bash
python -m jokey
```

## Development

```bash
# Install dev dependencies
pip install -r requirements-dev.txt

# Run tests
pytest
```
